Principais Melhorias e Novas Funcionalidades:
Monitoramento Avançado com Watchdog

Substitui a solução manual por uma biblioteca especializada

Detecta com precisão criações, modificações, exclusões e movimentações

Verificação de Integridade com Hash SHA-256

Calcula hashes dos arquivos para detectar alterações não autorizadas

Armazena hashes para comparação futura

Detecção de Padrões Suspeitos

Identifica possíveis arquivos maliciosos (executáveis, scripts)

Verifica conteúdo em busca de padrões conhecidos

Auditoria de Segurança Automatizada

Verifica permissões perigosas (escrita pública, executáveis com escrita)

Identifica arquivos em locais suspeitos (/tmp, /var/tmp)

Interface Aprimorada

Sistema de abas para melhor organização

Cores diferenciadas para logs (informação, alerta, erro)

Visualização em árvore com mais detalhes (tamanho, proprietário, etc)

Alertas Visuais e Sonoros

Popups para eventos críticos

Sons de alerta para chamar atenção

Controle Granular de Permissões

Permite configurar separadamente dono, grupo e outros

Visualização clara das permissões atuais

Sistema de Filas para Processamento Assíncrono

Não trava a interface durante operações longas

Processamento suave de eventos

Estatísticas e Relatórios

Contagem de vulnerabilidades encontradas

Resumo das verificações realizadas

Suporte Multiplataforma

Funciona tanto em Windows quanto Unix/Linux

Adapta-se às particularidades de cada sistema
